CSS Positions

  1. Static: by default eta thake
  2. Relative: current position relative e koto tuku change hbe and it will not leave the current spot.
  3. Absolute: nijer jaiga chere dise, gayeb hoye veshe ase
  4. Fixed: scroll korleo ekta jaigai e fixed hoye lege thake move hoi na
  5. Sticky: stick hoye thakbe upore, scroll korleo upore stick thakbe
